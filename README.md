# Resume Craft

Resume Craft is an AI-powered resume builder designed to help users create professional, ATS-friendly resumes quickly and efficiently. Leveraging the Groq API for AI-assisted content generation, users can generate resume summaries, work experience descriptions, skills, and other sections while maintaining full control over customization.

## Features

* AI-powered resume content generation using Groq API
* Professional and ATS-friendly resume layouts
* Real-time resume editing and preview
* PDF export functionality
* User-friendly and responsive interface
* Resume management dashboard
* Secure user authentication
